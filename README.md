# Deform

- ⚡ Lightning fast!
- ⚙ Multithreadeded
- 🌐 Works in worldspace
- 🔎 Custom editors and handles

A fully-featured deformer system for Unity. Compiled with Burst, multithreaded with the Job System and calculated with the Mathematics library.
